# SSDH

A paddlepaddle implementation for paper "Semantic Structure-based Unsupervised Deep Hashing" IJCAI-2018

## Requirements

- Python 3.8.18
- paddlepaddle 2.1.0
- cuda 11.2
- cudnn 8.2.1

## Demo
```
train:
$ python run.py --train
test:
$ python run.py --evaluate
```

## Dataset
- mir25_crossmodal.h5
