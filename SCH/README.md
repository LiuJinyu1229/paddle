# SCH
Source code for TPAMI paper "[Cross-Modal Hashing Method with Properties of Hamming Space: A New Perspective](https://ieeexplore.ieee.org/document/10506992/)"

## Requirements

- Python 3.8.18
- paddlepaddle 2.1.0
- cuda 11.2
- cudnn 8.2.1

## Demo
```
train:
$ python main.py --train
test:
$ python main.py
```

## Dataset
- mir_cnn_twt.mat
