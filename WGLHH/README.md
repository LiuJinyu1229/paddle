# WGLHH
Paddle implementation for the paper “Weighted Gaussian Loss based Hamming Hashing”

## Requirements

- Python 3.8.18
- paddlepaddle 2.1.0
- cuda 11.2
- cudnn 8.2.1

## Demo
```
train:
$ python train_transformer.py --train
test:
$ python train_transformer.py
```

## Dataset
- mir_cnn_twt.mat
